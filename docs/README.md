# Case Shield documentation index

This directory is the canonical coordination and architecture layer for Case Shield app development.

## Core docs
- `PLUGIN_ARCHITECTURE_STATUS.md` — current status of the plugin-based application structure.
- `AI_COORDINATION_LOG.md` — append-only operations log for AI and human contributors.
- `RESOURCE_CIRCUIT.md` — the shared awareness model, resource registry, and export rules.
- `TOOL_DRAWER_EXTERNAL_GOOGLE_APIS.md` — curated Google API roles for the project.
- `DRIVE_AND_GITHUB_STRUCTURE.md` — canonical file and folder structure across Drive and GitHub.
- `UPLOAD_PIPELINE_AND_CONNECTOR_POLICY.md` — onboarding rules for remotes, connectors, and mounts.

## Canonical rule
- GitHub holds the app-development skeleton and permanent docs.
- Google Drive holds live working coordination docs and presentation-layer records.
- Zorin is the active execution node.
